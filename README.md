## 구성   
* 회원가입  
  * 이메일, 비밀번호, 비밀번호, 확인, 이름, 성별, 나이, 사진  
* 로그인  
  * 이메일, 비밀번호  
* 비밀버호 찾기  
  * 이메일  
* 게시글 리스트 화면  
  * 게시글 아이템

  * 이메일, 이름, 성별, 나이, 사진
